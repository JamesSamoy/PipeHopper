Instructions to set-up and run Pipe Hopper Game:

1. Download and install node.js https://nodejs.org/en/
2. Install Express inside windows command prompt by typing "npm install express" and press enter.
3. Place all project files into a folder.
4. Open up windows command prompt and navigate to the root folder containing the files.
5. Start up the web server (server.js) by typing "node server.js" and press enter.
6. Open up your browser and type in "localhost:8080" and press enter.
7. When the game home page loads into the browser and the server is listening,
	you have successfully opened the game and are ready to play.
